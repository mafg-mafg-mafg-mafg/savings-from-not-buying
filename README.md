# Do not buy this product

**Do not buy this product** is a personal finance application designed to transform deprivation into motivation. Its primary goal is to help users visualize and quantify the money they save every time they consciously choose **not to make** an impulsive or unnecessary purchase.

## Key Features

1.  **Active Savings Logging:** Document every "frustrated expense" by assigning a product name and the cost avoided.
2.  **Animated Dashboard:** A central card displays the accumulated total savings. To make the experience more rewarding, the counter uses "count-up" animations and visual pulse effects whenever the savings increase.
3.  **Interactive Item Management:**
    *   **Frequency Tracker:** Each product in the list tracks how many times it has been avoided. Tapping an existing item increments its counter and automatically adds the amount to the total savings.
    *   **Intuitive Gestures:** Supports swipe-to-delete to remove products from the list and the database permanently.
4.  **Sensory Feedback (Haptic & Audio):** The app reinforces positive behavior through tactile vibrations and confirmation sounds every time a user registers a saving, creating a gratifying user experience.
5.  **Professional Data Portability (CSV):**
    *   **Export:** Generates detailed reports in `.csv` format with the default name *"Savings from NOT buying.csv"*. The report includes a product breakdown, subtotals, and a grand total row.
    *   **Import:** Allows users to restore or migrate the database from previously exported CSV files, ensuring that data is never lost.
6.  **Modern and Robust Design:**
    *   Built with **Material Design 3**, featuring a fixed color palette (green and white) that ensures a consistent visual identity regardless of system dynamic colors.
    *   Optimized for readability with bold text and precise alignments powered by *ConstraintLayout*.
7.  **Multilingual Support:** Fully localized in **English** and **Spanish**, adapting texts, currency formats, and navigation labels.

## Technical Specifications

*   **Language:** Kotlin.
*   **Persistence:** Room Database for fast and secure local storage.
*   **Architecture:** Jetpack Navigation Component with a Fragment-based architecture.
*   **Performance:** Extensive use of Kotlin Coroutines to maintain UI fluidity during file and database operations.

---

This is a minimalist, private, and powerful tool for those who want to take control of their finances by focusing on the savings derived from daily discipline.
